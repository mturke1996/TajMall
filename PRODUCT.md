# Fluxen - Mall Management System

## Product Purpose
Enterprise-grade financial management system for commercial malls (Taj Mall). Tracks revenue (rent, parking, services), expenses (security, maintenance, marketing), and provides real-time insights for decision makers.

## Users

### Primary Users
1. **Owner/Admin** - Full access, views all reports, manages users
2. **Accountant** - Manages transactions, generates reports
3. **Cashier** - Records daily transactions, views cashboxes
4. **Viewer** - Read-only access to reports and dashboards

### Usage Context
- Desktop: Primary usage in office for detailed work
- Mobile: Quick checks and transaction recording on the go
- Language: Arabic (RTL), with English support for international staff

## Brand Tone
Professional, trustworthy, modern Islamic finance aesthetic. Clean, spacious, premium feel. No clutter.

## Anti-References
- NO generic dashboard templates
- NO cluttered data tables
- NO corporate blue/grey boring colors
- NO complex permission systems that get in the way

## Strategic Principles
1. **Simplicity First** - Every feature must justify its existence
2. **Mobile-First** - Must work flawlessly on phones
3. **Connected Data** - Everything links: tenants → rent payments → reports
4. **Real-time** - Instant updates across all views
5. **Offline-Ready** - PWA capabilities for field work

## Core Features
1. Revenue tracking (rent, parking, services, ads)
2. Expense tracking (security, insurance, marketing, salaries)
3. Cashbox management (multiple treasuries)
4. Tenant directory with rent status
5. Employee directory with salary tracking
6. Financial reports (P&L, cash flow, trial balance)
7. Real-time dashboard with KPIs
8. Multi-user with simple role-based access

## Success Metrics
- Transaction recording under 10 seconds
- Page load under 2 seconds
- Mobile responsiveness score 95+
- Zero data entry errors through validation
